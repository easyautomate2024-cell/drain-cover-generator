# 排水溝蓋 STL ジェネレーター

ブラウザで動作する、お風呂の排水溝の蓋（ドレンカバー）専用 STL ジェネレーターです。

## 機能

- 🎛 **パラメトリック設計** — 外径・厚み・凹みサイズ・メッシュ寸法をスライダーで調整
- 👁 **ブラウザ 3D ビュー** — 回転・ズーム・パンで形状をリアルタイム確認
- 📐 **バウンディングボックス表示** — 全体寸法 (X / Y / Z) を即座に把握
- 🕳 **網目状メッシュ** — 凹み底面と側壁にケージ状の排水穴を生成
- 💾 **STL ダウンロード**（実装予定）

## 使い方

1. `index.html` をブラウザで開く（または GitHub Pages で配信）
2. 左サイドパネルのスライダーで寸法調整
3. 3D ビューで形状確認
4. STL ダウンロードボタンで保存

## 技術

- [Three.js](https://threejs.org/) (r128) によるブラウザ 3D 描画
- Pure HTML / CSS / JavaScript — ビルド不要

## 部品構造

- **外周リム** — ドーナツ形状のフラットな縁
- **凹み底** — 網目穴付きの円盤（排水機能）
- **凹み側壁** — 横リブ + 縦バーのケージ状メッシュ（排水機能）

凹み深さが蓋の厚みを超える場合、円筒状に下方向へ伸びる「ドレンチューブ」形状になります。
